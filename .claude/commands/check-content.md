# Проверить качество контента

## Использование
`/check-content [файл или секция]`

## Инструкции

1. Прочитай указанный файл или все файлы секции
2. Проверь по чеклисту:

### Frontmatter
- [ ] Все обязательные поля присутствуют (slug, sidebar_position, title, description, image, animation, obsIMG, free)
- [ ] slug — английский, kebab-case
- [ ] sidebar_position совпадает с числовым префиксом в имени файла
- [ ] description заполнен (не пустой)

### Контент
- [ ] Написан на русском
- [ ] Тон — разговорный, дружеский (не академический)
- [ ] Нет wiki-ссылок `[[...]]`
- [ ] Адмониции с русскими заголовками `:::tip[Заголовок]`
- [ ] H3 как основной уровень заголовков (не H1, не H2)
- [ ] Ключевые термины выделены `**жирным**`

### Техническое
- [ ] `npm run build` проходит без ошибок

3. Выведи отчёт с найденными проблемами и предложениями
