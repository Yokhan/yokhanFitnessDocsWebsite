# Правила Docusaurus

## Архитектура

- Docs подаются на root (`routeBasePath: '/'`)
- Blog отключён (`blog: false`)
- Сайдбар автогенерируется из файловой структуры (`autogenerated, dirName: '.'`)
- Порядок в сайдбаре определяется `sidebar_position` в frontmatter и `_category_.json`

## Статические ресурсы

Три директории подключены как static:
1. `static/` — логотипы, favicon
2. `docs/media/img/` — изображения контента
3. `docs/media/video/` — видео контента

## Стили

- Брендовый цвет: `#C8EA6A` (лайм) — не менять
- Шрифт H1: `nauriz`
- Шрифт H2-H6: `DelaGothicOne-Regular`
- Шрифт body: `IBMPlexSans-VariableFont`
- Тёмная тема code: Dracula, светлая: GitHub
- Table of contents скрыт (`display: none`)

## Проверка

После любых изменений в конфигах или контенте:
```bash
npm run build
```
Docusaurus выбрасывает ошибку на битые ссылки (`onBrokenLinks: 'throw'`).
