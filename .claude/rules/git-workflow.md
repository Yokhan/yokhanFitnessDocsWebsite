# Git workflow

## Коммиты

Формат: `тип: описание на русском`

Типы:
- `контент:` — новые/изменённые страницы в docs/
- `стиль:` — изменения CSS, шрифтов, визуала
- `конфиг:` — изменения docusaurus.config.js, package.json, sidebars.js
- `медиа:` — новые изображения/видео
- `инфра:` — CI/CD, .gitignore, настройки агента

Примеры:
- `контент: добавить страницу про разминку`
- `стиль: обновить размер заголовков H2-H3`
- `конфиг: обновить URL сайта`

## Ветки

- `main` — стабильная версия, не коммитить напрямую при работе через агент
- `claude/*` — рабочие ветки агента

## Что не коммитить

- `node_modules/`, `build/`, `.docusaurus/`
- IDE-файлы: `.vs/`, `.vscode/`, `.idea/`
- `.env*` файлы
